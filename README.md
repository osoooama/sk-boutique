# SK BOUTIQUE — متجر سارة كريشان

متجر أزياء نسائية عصرية — تصاميم أوروبية تُصنع محلياً بخامات إيطالية وفرنسية بمعايير الجودة الأوروبية.

## التشغيل

```bash
npm run dev
```

فتح [http://localhost:3000](http://localhost:3000).

## البناء

```bash
npm run build
```

## النشر

النشر التلقائي على **Cloudflare Pages** عند `git push`.
